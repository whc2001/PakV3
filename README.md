# PakV3
用于测试的剑网3旧版客户端资源包提取工具，可以从资源文件中读取连续的块并输出为独立的文件，但由于未能找到资源列表因此无法确认文件名与格式，包含了一个简易的工具类 (JX3FileTypeRecognizer.cs) 用来猜测文件格式但并不好用。

PakV3版客户端应该已经停止使用了所以丢上来存个档。
